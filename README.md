# Diffusion-Based Synthetic Defect Generation for Industrial Anomaly Detection

This project studies whether synthetic industrial defects generated by diffusion inpainting can improve downstream defect detection and segmentation.

## Current Stage

Stage 1 focuses on understanding the MVTec AD dataset and verifying that images and pixel masks are correctly aligned.
